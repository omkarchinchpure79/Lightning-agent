# MHT CET Category Code Legend
# Source: Official CET Cell CAP cutoff PDFs

## Gender Prefix
- G = General (open to all genders)
- L = Ladies (female candidates only)

## Seat Type Suffix Context
- H = Home University
- O = Other Than Home University
- S = State Level
- AI = All India Seat

## Reservation Categories (Government of Maharashtra)
- SC: Scheduled Caste (13%)
- ST: Scheduled Tribe (7%)
- VJ/DT: Vimukta Jati / De-notified Tribes (3%)
- NT1: Nomadic Tribe 1 (2.5%)
- NT2: Nomadic Tribe 2 (3.5%)
- NT3: Nomadic Tribe 3 (2%)
- OBC: Other Backward Class (19%)
- SBC: Special Backward Class (introduced 2024)
- EWS: Economically Weaker Section (10%)
- TFWS: Tuition Fee Waiver (separate quota for economically weaker)

## Notes
- SBC (Special Backward Class) codes (GSEBCS/H/O etc.) appear from 2024 onwards
- Category codes may vary slightly year-to-year — always verify against source PDF
- "PWDR" prefix = PwD common reserved pool; "DEF" prefix = Defence reserved pool
- All India quota rows use `category='AI'`, `is_all_india=1`
- Each AI quota row in the DB is one individual student allotment (not a per-branch cutoff)
- AI cutoff for a branch = minimum percentile among all AI rows for that branch/year/round
